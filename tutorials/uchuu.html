Uchuu Data Release 1

  • 2.1 trillion N-body simulation that uses Planck2015 cosmology
  • 2PB of data generated in the Aterui-II supercomputer
  • Reference: arXiv:1711.01453

Aterui-II

  • National Astronomical Observatory of Japan (NAOJ)
  • Launch Year: 2018
  • Cores: 40.200 SkyLake
  • Memory: 386 TB

Big Data

  • Goal: Allow large scale analysis of the results
  • Selection of 50 epocs
  • Transformation of the datasets to parquet
  • Interactive analysis using Python Notebooks and Spark
  • Uchuu github repo with examples
